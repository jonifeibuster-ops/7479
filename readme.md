## D&P Perfumum — многостраничный сайт‑магазин

Статический (без сервера) прототип интернет‑магазина парфюмерии с авторизацией, корзиной и отзывами.

### Страницы

- **`index.html`** — главная страница с презентацией бренда и популярными ароматами.
- **`catalog.html`** — каталог товаров с фильтром по категории и поиском.
- **`cart.html`** — корзина с изменением количества и расчётом суммы.
- **`login.html`** — вход покупателя и администратора.
- **`admin.html`** — панель администратора (модерация отзывов, номер WhatsApp).
- **`reviews.html`** — список отзывов и форма добавления отзыва.

Все страницы используют общий JavaScript (`app.js`) и стили (`styles.css`).

### Функционал

- **Авторизация (localStorage)**:
  - Покупатель: только e‑mail.
  - Администратор: e‑mail + пароль `admin123` (демо).
  - Статус входа и роль сохраняются в `localStorage`.
- **Корзина (localStorage)**:
  - Добавление товара из каталога и с главной.
  - Изменение количества, удаление, подсчёт общей суммы.
- **Отзывы (localStorage)**:
  - Покупатель может оставить отзыв (после входа).
  - Администратор одобряет / скрывает отзывы в админке.
  - На странице отзывов показываются только одобренные и не скрытые отзывы.
- **Кнопка WhatsApp**:
  - Плавающая кнопка внизу справа на всех страницах.
  - Номер задаётся в админке и хранится в `localStorage` (только в этом браузере).

### Как запустить

- Откройте файл `index.html` двойным щелчком в браузере **или**:
  - Поднимите любой статический сервер (например, VS Code Live Server, `npx serve` и т.п.) в папке проекта.
